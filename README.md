# Terraform Dev

AI-native terminal REPL for HCP Terraform.

![Demo](demo.gif)

## Quick start

```bash
git clone https://github.com/rchandnaWUSTL/terraform-dev-terminal.git
cd terraform-dev-terminal
make install
terraform-dev
```
